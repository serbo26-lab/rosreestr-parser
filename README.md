# Локальный парсер справочной информации Росреестра

Программа для Windows, которая помогает собирать справочную информацию Росреестра и сохранять её в Excel.

Подходит для работы с:

- многоквартирными домами;
- реестрами для ОСС;
- квартирами, помещениями и машино-местами;
- земельными участками СНТ;
- старыми реестрами, которые нужно проверить или обновить.

Программа работает на вашем компьютере.  
Все результаты сохраняются рядом с программой в папку `output`.

---

## Как скачать и запустить

1. Откройте страницу:

   https://github.com/serbo26-lab/rosreestr-parser/releases

2. Скачайте файл:

   ```text
   rosreestr_parser.zip
   ```

3. Распакуйте архив в отдельную папку.

4. Откройте файл:

   ```text
   RosreestrParser.exe
   ```

5. В программе выберите нужный раздел на вкладке **Главная**.

---

## Что умеет программа

### МКД / ОСС

Программа может:

- собрать сведения по квартирам и помещениям дома;
- подготовить реестр для ОСС;
- собрать кадастровые номера;
- собрать полные сведения по объектам;
- сопоставить новый реестр со старым;
- перенести данные из старого реестра, если они совпадают по праву.

Поддерживаются форматы:

- Burmistr;
- Roskvartal.

### СНТ / земельные участки

Программа может:

- собрать сведения по участкам СНТ;
- искать участки по названию товарищества и номерам;
- учитывать актуальные и погашенные участки;
- записывать найденные здания на отдельный лист;
- продолжать работу с места остановки;
- перепроверять участки, которые не были найдены;
- обновлять старый рабочий реестр СНТ без удаления пользовательских колонок.

---

## Как пользоваться

Откройте программу и выберите задачу на вкладке **Главная**:

```text
ОСС / МКД — собрать реестр
ОСС / МКД — сопоставить со старым реестром
СНТ / земли — собрать участки
СНТ — актуализировать рабочий реестр
```

Дальше программа откроет нужную вкладку.

Обычно достаточно заполнить основные поля и нажать кнопку **Старт**.

Редкие настройки спрятаны в раскрываемые блоки. Их не нужно менять без необходимости.

---

## Вход в Росреестр

Программа открывает обычный браузер.

Если Росреестр попросит войти, пройти капчу, подтвердить вход по SMS или QR-коду, это нужно сделать вручную в браузере.

После входа нажмите в программе:

```text
Продолжить после входа/капчи
```

Программа сама не вводит логин, пароль, SMS-код и капчу.

---

## Где искать результат

После работы программа создаёт Excel-файлы в папке:

```text
output
```

Обычно создаются два файла:

```text
result_...xlsx
log_result_...xlsx
```

Главный файл:

```text
result_...xlsx
```

Это основной результат.

Служебный файл:

```text
log_result_...xlsx
```

Он нужен для проверки, ошибок, спорных случаев и подробностей поиска.

Если выполнялось сопоставление реестров, файлы будут называться примерно так:

```text
matched_...xlsx
log_matched_...xlsx
```

## Что важно не удалять

Рядом с программой могут появиться папки:

```text
output
state
profiles
profile_rosreestr
updates
```

Что это значит:

- `output` — готовые Excel-файлы;
- `state` — данные для продолжения работы с места остановки;
- `profiles` — сохранённые настройки;
- `profile_rosreestr` — данные браузера для входа в Росреестр;
- `updates` — скачанные обновления.

Если не уверены, лучше эти папки не удалять.

---

## Обновление программы

Проверить новую версию можно внутри программы:

```text
О программе → Проверить обновление
```

Если новая версия доступна, программа предложит скачать или установить обновление.

При обновлении не должны удаляться ваши результаты, настройки и профили.

---

## Важно

Программа работает только со справочной информацией, которую пользователь открывает через браузер.

Программа:

- не заказывает платные выписки;
- не отправляет заявления;
- не выполняет юридически значимые действия;
- не обходит капчу;
- не заменяет официальную выписку из ЕГРН.

Для суда, сделок и официальных процедур нужно использовать официальные документы и выписки.

## Если программа не запускается

1. Убедитесь, что архив распакован, а не открыт прямо из ZIP.
2. Перенесите папку программы в простое место, например `C:\RosreestrParser`.
3. Если Windows показывает предупреждение безопасности, нажмите `Подробнее` → `Выполнить в любом случае`.
4. Если после распаковки файл RosreestrParser.exe исчез, проверьте журнал защиты Windows. Возможно, файл был помещён в карантин как неизвестная новая программа.
5. Если программа пишет, что не найден браузер, установите Google Chrome.
6. Если появляется ошибка VCRUNTIME140.dll, VCRUNTIME140_1.dll или MSVCP140.dll, установите Microsoft Visual C++ Redistributable 2015–2022 x64.
7. Если появляется сообщение “Это приложение не может работать на вашем ПК”, проверьте версию Windows. Программа рассчитана на Windows 10/11 64-bit.
