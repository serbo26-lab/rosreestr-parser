# Локальный парсер справочной информации Росреестра

Локальная Windows-программа для работы со справочной информацией Росреестра.

Программа помогает собирать сведения по объектам недвижимости, готовить реестры для ОСС/МКД, работать с земельными участками СНТ и актуализировать ранее собранные реестры.

## Быстрый запуск обычной версии

1. Откройте раздел **Releases**:

   https://github.com/serbo26-lab/rosreestr-parser/releases

2. Скачайте файл:

   ```text
   rosreestr_parser.zip
   ```

3. Распакуйте архив в отдельную папку.
4. Запустите:

   ```text
   RosreestrParser.exe
   ```

5. Выберите нужную вкладку программы, заполните поля, выставьте настройки и нажмите `Старт`.
6. В открывшемся браузере авторизуйтесь вручную, когда Росреестр попросит вход. Рекомендуется авторизация через QR-код.
7. После успешного входа нажмите в программе `Продолжить после входа/капчи`.

## Если вы скачали исходники

- `00_SETUP.bat` — установка Python-окружения и Playwright для запуска из исходников;
- `01_RUN.bat` — запуск программы из исходников;
- `02_BUILD_EXE.bat` — сборка portable EXE-архива.

## Для чего нужна программа

### ОСС / МКД

- парсинг квартир и помещений по диапазону номеров;
- дополнительные номера вроде `9А`, `15Б`, `27/1`;
- сбор реестра для ОСС;
- отдельные форматы выгрузки:
  - Burmistr;
  - Roskvartal;
- полный парсинг карточки «Сведения об объекте»;
- сбор только кадастровых номеров;
- сопоставление нового реестра ОСС со старыми Excel/Word-реестрами.

### СНТ / земельные участки

- парсинг земельных участков по городу, названию СНТ и кадастровым кварталам;
- поддержка актуальных и погашенных участков;
- отдельный лист для зданий, найденных по номеру участка;
- продолжение с места после остановки;
- перепроверка не найденных участков;
- актуализация старого рабочего реестра СНТ с сохранением пользовательских колонок.

## Основные вкладки

Порядок вкладок в программе:

```text
Парсинг Росреестра ОСС
Сопоставление реестра ОСС
WIKI ОСС
Парсинг Росреестра СНТ
Актуализация реестра СНТ
WIKI СНТ
О программе
```

Подробная инструкция находится внутри программы в разделах **WIKI ОСС** и **WIKI СНТ**.

## Обновления

Программа проверяет наличие новой версии через GitHub:

```text
О программе → Проверить обновление
```

## Что сохраняется локально

Программа работает локально на компьютере пользователя.

Обычно рядом с программой создаются папки:

```text
output/
state/
profiles/
profile_rosreestr/
updates/
```

Назначение:

- `output/` — результаты парсинга и log-файлы;
- `state/` — файлы прогресса для продолжения работы;
- `profiles/` — сохранённые настройки программы;
- `profile_rosreestr/` — профиль браузера;
- `updates/` — скачанные архивы обновлений.

## Важно о работе с Росреестром

Программа открывает обычный видимый браузер и работает с открытой пользователем страницей Росреестра.

Программа:

- не обходит капчу;
- не вводит логин, пароль и SMS-код вместо пользователя;
- не заказывает платные выписки;
- не отправляет заявления;
- не выполняет юридически значимые действия.

Авторизация, SMS-коды и капча вводятся только пользователем в браузере. Рекомендуется авторизация через QR-код.
