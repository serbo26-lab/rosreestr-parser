# Логика парсера V56

## Основной проход по квартирам

Парсер перебирает адреса по диапазону квартир:

- `г Город ул Улица д 1 кв 1`
- `г Город ул Улица д 1 кв 2`
- и т.д.

Если включена галочка вариантов адреса, для каждой квартиры пробуются варианты:

- `кв 1`
- `кв №1`
- `кв № 1`
- `№1`
- `№ 1`
- `1`

Выбираются только результаты того же дома и той же квартиры. Погашенные объекты пропускаются, актуальные — записываются.

## Поля Excel

Заполняются колонки шаблона:

- Номер помещения (обяз.)
- Площадь помещения, м2 (обяз.)
- Кадастровый номер
- Признак совместной собс. ("да" для сов. собс.)
- Доля
- Муницип. собс.
- Номер регистрации (обяз.)
- Дата регистрации
- Фамилия / Наименование организации (обяз.)
- Имя (обяз. для физ лиц)
- Отчество
- Адрес (местоположение)

## Если прав нет

Если актуальный объект найден, но сведений о правах нет:

- Номер помещения — из адреса/карточки
- Площадь — из Росреестра
- Кадастровый номер — из Росреестра
- Признак совместной собс. — пусто
- Доля — `1/1`
- Номер регистрации — `.`
- Дата регистрации — пусто
- Фамилия / Имя / Отчество — `.`

## Если прав несколько

Одна запись права = одна строка Excel.

- `Собственность` → доля `1/1`
- `Общая долевая собственность` → доля пропорционально количеству записей, например 2 записи = `1/2`
- если есть слово `совместная` → признак совместной `да`, доля `1/1`

`№` и `от` в Excel не пишутся: номер и дата права записываются в разные поля чистыми значениями.

## Муниципальная собственность

- `Частная` → колонка муниципальной собственности пустая
- `Муниципальная` или `Государственная` → `да`

## Дополнительный проход по голому адресу дома

Если включена галочка «Добавить помещения без номеров/нежилые», после основного диапазона квартир парсер ищет по адресу дома без номера квартиры. Например:

`г Город, ул Улица, д 1`

Он берёт кандидаты того же дома, у которых в строке результата нет обычного маркера квартиры `кв`. Это нужно для нежилых помещений и объектов без номера помещения.

## Файл результата

Если включено автоимя, итоговый файл сохраняется в папку output с именем по адресу дома и диапазону квартир. Имя очищается от точек, запятых и запрещённых символов Windows.
