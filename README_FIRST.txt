Rosreestr Parser
================

Быстрый запуск релизной EXE-версии

1) Скачайте `rosreestr_parser.zip` из последнего GitHub Release.
2) Распакуйте архив в отдельную папку, например:

   C:\rosreestr_parser

3) Запустите:

   RosreestrParser.exe

4) Выберите нужную вкладку программы и запустите парсинг.
5) В открывшемся браузере вручную войдите через Росреестр/Госуслуги/
6) После успешного входа нажмите в программе «Продолжить после входа/капчи».

Если вы скачали исходники, а не релизную EXE-версию:
- `00_SETUP.bat` устанавливает окружение;
- `01_RUN.bat` запускает программу из исходников;
- `02_BUILD_EXE.bat` собирает portable EXE-архив.
